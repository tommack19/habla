# A2 Elementary

A2 teaches learners to build real sentences and handle common daily situations.

Learners expand vocabulary for restaurants, travel, work, hobbies, directions, shopping, and small talk. Grammar adds more present-tense verbs, common irregular verbs, `gustar`, simple past-tense exposure, direct object basics, and question formation.

By the end of A2, learners should be able to describe routines, talk about preferences, ask and answer practical questions, and manage basic conversations while traveling or visiting family.
