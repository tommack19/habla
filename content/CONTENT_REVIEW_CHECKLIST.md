# Habla Content Review Checklist

Version 1.0 — Frozen production baseline

Use this gate before calling any A1 lesson production-ready.

## 1. Mission

- [ ] The lesson can be summarized as “Today you’ll learn to…” in one sentence.
- [ ] The mission describes something a learner might genuinely do.
- [ ] Every section supports that mission.

## 2. Story continuity

- [ ] Carlos introduces the situation naturally.
- [ ] The scene follows logically from the previous lesson.
- [ ] At least three previously learned words or phrases return naturally.
- [ ] The lesson does not depend on language the learner has not encountered.

## 3. Vocabulary

- [ ] The lesson contains 15–30 useful words and chunks.
- [ ] Most vocabulary is Tier 1; Tier 2 appears only when the situation requires it.
- [ ] Words are grouped by communication purpose, not alphabetically.
- [ ] Every item answers: “Will this help someone speak today?”
- [ ] Decorative or rare topic vocabulary has been removed.

## 4. Essential phrases

- [ ] Phrases sound natural across the Spanish-speaking world.
- [ ] Each phrase has a clear use case and realistic example.
- [ ] The learner practices complete chunks, not isolated translations only.
- [ ] Neutral Spanish is the default.
- [ ] Regional variants appear only as optional, useful context.

## 5. Grammar

- [ ] The lesson teaches exactly one new grammar concept.
- [ ] Learners see the pattern in context before reading the explanation.
- [ ] The explanation is short, practical, and free of unnecessary terminology.
- [ ] Examples directly support the lesson mission.
- [ ] The practice prompt asks the learner to say something true or useful.

## 6. Listening and speaking

- [ ] Listening tasks use language taught in this or earlier lessons.
- [ ] Every listening answer exactly matches one option.
- [ ] Distractors are plausible and meaningfully different.
- [ ] Speaking moves from listen → repeat → recall → personalize.
- [ ] Pronunciation guidance focuses only on sounds or rhythm that matter now.

## 7. Conversation

- [ ] The mini conversation has a clear setting and goal.
- [ ] Carlos stays at the learner’s current level.
- [ ] The learner produces language instead of only selecting answers.
- [ ] The exchange includes a natural opener, response, follow-up, and closing when appropriate.
- [ ] The conversation feels like time spent with Carlos, not a scripted test.
- [ ] When multiple equally valid choices teach the same objective, the learner can choose.
- [ ] Carlos responds specifically to the learner’s choice before branches rejoin the story.
- [ ] Optional world-building makes the setting feel alive without becoming required vocabulary.

## 8. Culture

- [ ] The cultural note is one short, useful paragraph.
- [ ] It helps the learner behave or speak more naturally.
- [ ] It avoids stereotypes and unnecessary generalizations.

## 9. Challenge and success

- [ ] The Carlos Challenge uses the lesson’s new language and recycled language.
- [ ] Success criteria are observable and achievable.
- [ ] Completion represents a real communication success.
- [ ] Carlos celebrates communication before correcting one important issue.

## 10. Review and quality

- [ ] Review feels like continued conversation, not repetition.
- [ ] Multiple-choice options are unique and plausible.
- [ ] Spanish punctuation, accents, translations, and examples are correct.
- [ ] No content invents learner history, progress, or personal details.
- [ ] JSON matches `content/schema.md` and loads in the app.
- [ ] The lesson passes the golden rule.
